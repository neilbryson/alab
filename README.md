# alab

A sample front-end web application written in Blazor WebAssembly (C#).

## System requirements
* [.NET Core 3.1 SDK](https://dotnet.microsoft.com/download/dotnet-core/3.1)

## Running the app
```bash
cd src
dotnet run

## Watch mode
dotnet watch run
```

This will start a server at http://localhost:5000.
